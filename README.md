# RemoveActivityCard

[![By bonjourdev](https://img.shields.io/badge/By%20bonjourdev-ff0d86)](https://github.com/bjrdev) [![Latest Version](https://img.shields.io/github/v/release/bjrdev/RemoveActivityCard?display_name=tag&label=Latest%20Version&color=blue)](https://github.com/bjrdev/RemoveActivityCard/releases/latest) [![Download](https://img.shields.io/badge/Download-brightgreen)](https://github.com/bjrdev/RemoveActivityCard/releases/latest/download/RemoveActivityCard.bonjourdev.theme.css)

A simple BetterDiscord theme that removes the **"Activity" card** from the member list sidebar in Discord.

## 📦 Installation

1. Download or copy the theme file.
2. Place it in your BetterDiscord plugins folder:
```
%appdata%/BetterDiscord/themes
```
3. Enable **RemoveActivityCard** in BetterDiscord settings.

## 🛠️ Usage

Once enabled, the activity cards will automatically be hidden from the member list.
<br>No configuration needed.
